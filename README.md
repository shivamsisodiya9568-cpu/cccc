# RS Clinic Website

A large 7-page responsive clinic website for **RS Clinic, Agra**.

## Pages
- `index.html` — Home page
- `about.html` — About clinic
- `services.html` — Services / departments
- `doctors.html` — Doctor and care team
- `appointment.html` — Appointment booking form
- `gallery.html` — Gallery and testimonials
- `contact.html` — Contact details and map

## Main Features
- Responsive design for PC, mobile and tablet
- Liquid-glass style UI
- Smooth scroll reveal animations
- Mobile hamburger menu
- WhatsApp and call buttons
- Appointment form demo
- AI clinic help widget
- Google map embed for Agra
- Separate `style.css` and `script.js`

Open `index.html` in your browser. For GitHub Pages, upload all files together in one repository.
